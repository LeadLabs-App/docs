## Running a Canvas

1. Visit a **prospect’s LinkedIn profile**.
2. Click the **Run Canvas** button.
3. A list of all canvases in your library will appear.
4. Select the canvas you want to use (e.g., _Multi-Signal Outreach_).

Within a few seconds, LeadLabs will run the entire canvas automatically.
